# 🐍 VS Code + Python: Shortcuts Esenciales

## 💻 Atajos de Interfaz (Generales)
| Acción | Shortcut (Windows) |
| :--- | :--- |
| **Paleta de Comandos** (Busca todo) | `Ctrl + Shift + P` |
| **Abrir archivo rápidamente** | `Ctrl + P` |
| **Abrir/Cerrar barra lateral** | `Ctrl + B` |
| **Nueva Terminal** | `Ctrl + Shift + ñ` (o backtick `` ` ``) |
| **Zen Mode** (Sin distracciones) | `Ctrl + K`, luego `Z` |

---

## 🐍 Atajos Específicos de Python
Para que estos funcionen, asegúrate de tener instalada la extensión oficial de **Python** en VS Code.

| Acción | Shortcut |
| :--- | :--- |
| **Ejecutar archivo actual** | `F5` (Debug) o `Shift + Enter` |
| **Seleccionar Intérprete** | `Ctrl + Shift + P` > *Python: Select Interpreter* |
| **Ir a Definición** (Función/Clase) | `F12` |
| **Ver Definición rápida** (Pop-up) | `Alt + F12` |
| **Renombrar Símbolo** (En todo el archivo) | `F2` |
| **Formatear documento** (Autopep8/Black) | `Shift + Alt + F` |

---

## ⚡ Edición de Código (Pro)
Estos te harán programar mucho más rápido:

* **Multicursor:** `Alt + Clic` (para escribir en varios lados a la vez).
* **Duplicar línea:** `Shift + Alt + ↓` o `↑`.
* **Mover línea entera:** `Alt + ↓` o `↑`.
* **Comentar bloque de código:** `Ctrl + /`.
* **Expandir/Contraer selección:** `Shift + Alt + →` o `←`.

---

## 🛠️ Snippets Útiles
Escribe esto y presiona `Tab`:

1.  `ifmain`: Crea la estructura `if __name__ == "__main__":`.
2.  `def`: Estructura para definir una función.
3.  `class`: Estructura para definir una clase.
4.  `try`: Bloque de manejo de excepciones `try/except`.

---

> **Tip:** Si quieres cambiar alguno de estos, ve a **File > Preferences > Keyboard Shortcuts** (`Ctrl + K` `Ctrl + S`).
